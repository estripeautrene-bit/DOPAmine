## The Three-Step Journey

This is what DOPAmine does to a person. In order. Always in this direction.

```
MINDSET  →  MENTAL RESILIENCE  →  MAGNETIC CONFIDENCE
```

**Step 1 — Mindset**
Where the user starts. They are growth-oriented, already moving,
already investing in themselves. They know mindset matters.
They just don't have a daily practice that sticks.
DOPAmine meets them here.

**Step 2 — Mental Resilience**
What the daily practice builds. 3 small wins. 2 minutes. Every day.
The brain learns to notice what's working. The filter rewires.
Setbacks land differently. Recovery is faster. The floor gets higher.
This is the mechanism. This is what DOPAmine does.

**Step 3 — Magnetic Confidence**
Where the user arrives. The confidence that others feel in a room.
The certainty that comes from accumulated evidence of progress.
Not performed confidence. Earned confidence.
This is the destination. This is what DOPAmine sells.

## The Brand Voice — Three Rules That Never Break

**Rule 1 — Positives only.**
Every sentence affirms, celebrates, or looks forward.
Zero negations. Zero absence framing.

**Rule 2 — Human language only.**
No ratios. No percentages. No framework jargon.
Write the way a person talks, not the way a whitepaper reads.

**Rule 3 — Science as proof, not headline.**
Lead with the feeling or the destination.
Science follows as the reason to believe.

## The Banned List — Permanent

These words and patterns are banned across all channels, forever.

| Banned | Replace with |
|--------|-------------|
| Journal / journaling | Daily practice / moment capture |
| Gratitude | Evidence of progress |
| Transform / transformation | Build / develop / strengthen |
| Amazing / incredible / life-changing | Specific and earned descriptions |
| Toxic positivity | — (never reference this concept) |
| Mental health | Mental performance / mental resilience |
| "Studies show..." as a headline | Lead with the feeling, cite science in the body |
| Any ratio or percentage as copy | Human language only |
| "Are you ready to...?" | Direct statements only |
| Great job / proud of you / keep it up | Specific, earned observations |

## The North Star Check

Before any content ships — on any channel — run it through this:

1. Does it serve the journey? Mindset → Mental Resilience → Magnetic Confidence?
2. Does it speak to the right person? Growth-oriented, already moving?
3. Is it in human language? No jargon, no ratios, no framework speak?
4. Does every sentence affirm, celebrate, or look forward?
5. Does it connect back to the one sentence?

If yes to all five — it ships.
If no to any one — it goes back.
